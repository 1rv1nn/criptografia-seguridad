# Uso
1. Abre una terminal en la carpeta del script:
   - `cd "Cifrado Decimado"`
2. Ejecuta el script indicando el archivo de entrada:
   - `python3 CifradoDecimado.py file4.lol`

## Notas
- La salida siempre se llama `output.pdf` y se sobrescribirá si ya existe.
- El desplazamiento es fijo: resta 50 a cada byte (módulo 256).
